FOODIST(Android)

 

설치방법

  0. 'Download for Andoid' 버튼을 눌러 설치파일 다운로드   

  1. 알림창에서 다운로드 파일 선택(실행)하여 설치 시작

  2. 보안상 이유로 설치가 차단되었을 경우, 설정을 눌러 진행

  3. 출처를 알 수 없는 App 또는 Play프로텍트에 의해 차단됨이란 내용으로 차단되었을 경우,

     적용 또는 무시하고 설치을 눌러 진행

  4. 설치버튼이 나올 때 까지 다음버튼을 눌러 진행

  5. 설치버튼을 눌러 설치 진행

  6. 설치완료

  7. App 시작

 

 

 

기능목록

  1. 발주관리 : 식재주문,발즈,취소

   2. 재고관리 : 식재료비율

   3. 클레임관리

   4. 영업관리 : 매출,입금,채권확인

   5. 승인관리 : 팀장,기획

   6. 여신관리 : 임시여신신청,승인

   7. 상품정보 : 검색

   8. 이미지전송 : (클레임,위생,CS) 촬영,전송

   9. 게시판

   10. efoodist

 

기타사항

    1. 권장사양 : Android 9.0 이상

    2. 원격지원 : 이지모바일
